Sheffield geeks

We're a group of Sheffield-based geeks who like to hang out on IRC and IRL. This page lists local upcoming events, meetups, and places to hang out or work.

You can add an event to the calendar through Open Tech Calendar. To add anything else open an issue on GitHub or just checkout the code for this page and send a pull request.
